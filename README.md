# Análisis de vehículos usados

Esta aplicación web fue desarrollada con Streamlit para explorar un conjunto de datos de anuncios de vehículos usados.

## Funcionalidades

- Visualizar un histograma de precios de vehículos.
- Visualizar un gráfico de dispersión entre kilometraje y precio.
- Explorar datos mediante gráficos interactivos con Plotly Express.

## Tecnologías utilizadas

- Python
- Pandas
- Plotly Express
- Streamlit

## Cómo ejecutar el proyecto

```bash
pip install -r requirements.txt
streamlit run app.py
